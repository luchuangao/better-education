# 允许批评

教育中一个显著特征是追求确定性，而这种过度追求往往会带来许多弊端。要规避这些弊端，关键是要学会接受批评，尤其作为家长，既要接受孩子的批评，也要勇于自我批评。

然而，批评在我们的文化中常常被忽视。一个最明显的例子就是，许多家长习惯用“听话”“乖”来衡量孩子的好坏。批评其实是独立思考的唯一途径；不允许批评，就意味着我们可能一直走在错误的道路上却毫无觉察，甚至会用手中那一点权力去无故为难孩子。

如果我们具体拆解教育追求确定性所带来的问题，主要包括以下几个方面：
*   **过度应试化**
*   **强调服从，压抑独立思考**
*   **失败成本极高，容错空间小**
*   **创造力训练不足**
*   **情绪与心理教育缺失**
*   **过度同质化，抹杀个性**

在这样的环境下，“听话”的孩子被认可，而“不听话”的孩子则要承担巨大的代价。他们往往需要耗费一生去消化这种排斥，容易陷入受害者心态，因为不服从主流，就仿佛被整个社会抛弃，鲜有人认同，走的是一条孤独而艰难的英雄之路。

对我自己来说，我承认一身毛病，也并不介意孩子指出我的问题。毛病是事实，无需隐藏。我们能做的，就是主动承认并改正，而不是掩饰或狡辩。

既然我们很难改变教育的整体标准，那么能做的就是强化自身：首先不在家里沿用传统权威的教育方式对待孩子；其次，尽力为孩子争取更宽松的成长环境，包括积攒一定的资金，将来如果孩子需要，也能支持他出国开阔视野。

这里顺便提一句：真正的爱国，不是没见过世界却整天喊打喊杀，而是出去看过之后，依然选择回来建设家园的人。他们不会对国家的问题视而不见，而是敢于批评、愿意推动改善。如果国家的问题不让批评，即便回来的人也会选择离开，因为经历过风霜的成年人只筛选不教育。
