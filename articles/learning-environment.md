# 学习环境

要让孩子更好地成长与学习，环境起着至关重要的作用。相比外部环境，家庭氛围的影响更为深远，在潜移默化中学习，远比死记硬背有效得多。今天想分享一下我们家是如何营造学习环境的。

父母是孩子最初的榜样。有一个爱学习的家长，孩子自然会模仿大人的一举一动。作为家长，首先要清楚当前阶段的学习重点。比如我儿子正在上小学，现阶段核心是语数外，再配合运动与兴趣爱好。如果要排序，我家把运动放在第一位，兴趣爱好第二，语数外随后。

运动和兴趣其实是孩子天生就容易投入的领域：运动释放精力、促进身体发育，但现在不少家长把孩子关在家里，缺乏户外活动，这对成长非常不利；而像唱歌、画画、跳舞这类兴趣爱好，孩子通常不需要催促就会主动去做。只要这两方面能做好，就可以把重心转到“最少必要知识”，也就是语数外上。

关键是把知识学习变成乐趣：

语文与英语都是语言学科，重点在于交流与表达。我家强调大量阅读，每天读书的时间至少比写作业多一倍。哪怕作业写不完，也优先保证阅读。英语学习类似，除了读，还通过看动画、听听力、日常简单对话来营造语境。寒假期间，我们会用免费的英文听力软件，找孩子喜欢的动画音频给他“磨耳朵”。每天虽有屏幕时间，但听比看更重要，对提升英语更有效。

数学比较抽象，很多家长容易陷入“拼命做题”的误区。其实更好的方式是依托孩子的好奇心，在生活中建立数学兴趣。比如算零花钱、坐电梯时数楼层、玩扑克算点数、猜硬币正反面……把抽象概念转化成日常可见的事物，逐步构建数学思维。

这就是我们家构建学习环境的方式。不需要高昂花费，人人都能做到，却可能是最朴素、也最有效的学习方法。
